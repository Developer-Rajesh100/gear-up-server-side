1) জাভাস্ক্রিপ্ট এবং নোডজের মধ্যে পার্থক্য
➤ 
জাভাস্ক্রিপ্ট একটি ভাষা যা ব্রাউজার দ্বারা লোড করা নথির অংশ হিসাবে ওয়েব ব্রাউজারগুলির মধ্যে চলে এবং একটি ক্লায়েন্ট-সাইড ডেভেলপমেন্ট ভাষা হিসাবে ব্যবহৃত হয়। এটি পৃষ্ঠাগুলির আচরণ প্রদান করে। HTML এর মত এটি শব্দার্থিক গঠন এবং CSS এর চেহারা এবং অনুভূতি প্রদান করে।
NodeJs হল একটি ওপেন সোর্স, ক্রস-প্ল্যাটফর্ম পরিবেশ যা আপনাকে সার্ভার-সাইড অ্যাপ্লিকেশন তৈরি করতে দেয়।

(JavaScript is a language that runs inside web browsers as part of the documents loaded by the browser and is used as a client-side development language. It provides the behavior of the pages. Like HTML it provides the semantic structure and CSS the look and feels.
NodeJs is an open-source, cross-platform environment that allows you to create server-side applications. )

2) আপনার কখন nodejs ব্যবহার করা উচিত এবং কখন MongoDB ব্যবহার করা উচিত?
➤ 
Node.js হল একটি একক-থ্রেডেড, ওপেন-সোর্স, ক্রস-প্ল্যাটফর্ম রানটাইম পরিবেশ দ্রুত এবং মাপযোগ্য সার্ভার-সাইড এবং নেটওয়ার্কিং অ্যাপ্লিকেশন তৈরির জন্য। এটি V8 জাভাস্ক্রিপ্ট রানটাইম ইঞ্জিনে চলে, এবং এটি ইভেন্ট-চালিত, নন-ব্লকিং I/O আর্কিটেকচার ব্যবহার করে, যা এটিকে দক্ষ এবং রিয়েল-টাইম অ্যাপ্লিকেশনের জন্য উপযুক্ত করে তোলে।

MongoDB হল একটি অবজেক্ট-ওরিয়েন্টেড, সরল, গতিশীল এবং মাপযোগ্য NoSQL ডাটাবেস। এটি NoSQL ডকুমেন্ট স্টোর মডেলের উপর ভিত্তি করে তৈরি। ডেটা অবজেক্টগুলি একটি সংগ্রহের ভিতরে পৃথক নথি হিসাবে সংরক্ষণ করা হয়।

(Node.js is a single-threaded, open-source, cross-platform runtime environment for building fast and scalable server-side and networking applications. It runs on the V8 JavaScript runtime engine, and it uses event-driven, non-blocking I/O architecture, which makes it efficient and suitable for real-time applications.

MongoDB is an object-oriented, simple, dynamic, and scalable NoSQL database. It is based on the NoSQL document store model. The data objects are stored as separate documents inside a collection.)

3) SQL এবং NoSQL ডাটাবেসের মধ্যে পার্থক্য।
➤ 
SQL বা স্ট্রাকচার্ড কোয়েরি ল্যাঙ্গুয়েজ হল রিলেশনাল ডাটাবেস ম্যানেজমেন্ট সিস্টেমের জন্য সবচেয়ে সাধারণ এবং জনপ্রিয় প্রোগ্রামিং ভাষা। এটি কাঠামোগত ডেটা এবং কৌশলগত বিশ্লেষণের জন্য ডেটা নিষ্কাশন, সঞ্চয়, সন্নিবেশ, মুছে ফেলা, আপডেট এবং পরিচালনা করার জন্য ডিজাইন করা একটি ভাষা।

NoSQL ডাটাবেস ডেটা সঞ্চয় এবং পুনরুদ্ধারের জন্য একটি প্রক্রিয়া সরবরাহ করে যা টেবুলার আকারে ব্যতীত মডেল করা হয়। এটি কার্ল স্ট্রোজ দ্বারা 1998 সালে একটি অ-রিলেশনাল ডাটাবেস নামে পরিচিত হয়েছিল। এখন, এটি শুধুমাত্র SQL এর জন্য দাঁড়িয়েছে।

(
    SQL or the Structured Query Language is the most common and popular programming language for the relational database management system. It is a language designed to extract, store, insert, delete, update and manage data for structured data and strategic analysis. 

    NoSQL database provides a mechanism for storage and retrieval of data that is modeled other than in tabular form. It was introduced by Carl Stroz in the year 1998 called a Non-relational database. Now, it stands for Not only SQL.


)
